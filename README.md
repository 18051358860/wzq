一、方案标题
------------
  在线宠物喂食器系统
二、应用场景
---------------
当铲屎官遇到疯狂加班，主子们在家挨饿时；
当外出出差多天；
家里有一个在线宠物喂食是非常有必要的。
三、功能
----------------
1、远程喂食功能
  将设备接入互联网，通过微信小程序投喂。
2、水量食量监测功能
  当食量或者水量低于设定值时，报警提示投喂。
3、定时定量投喂
  按照宠物的饮食规律按一定时间，一定量投喂。
4、语音识别功能
  当宠物发出不正常声音时，通过微信小程序提醒铲屎官。
